# Sistema de organización y automatización para UADY

Estas son ideas iniciales del sistema. El enfoque todavía puede cambiar cuando se defina mejor el alcance final.

## 1. Descripción

El sistema busca resolver problemas de organización dentro de la escuela, específicamente en procesos que hoy son repetitivos, manuales y administrativos.

La idea es construir una plataforma que se convierta en el núcleo de la administración de tareas:

- contables;
- administrativas;
- burocráticas;
- legales;
- escolares;
- documentales.

El sistema debe incluir:

- flujos automatizados;
- una interfaz para operar tareas y consultar información;
- un asistente de IA con contexto del sistema;
- herramientas para buscar documentos, explicar flujos y apoyar acciones operativas.

El usuario, especialmente el personal administrativo, debería poder interactuar con la IA para:

- encontrar documentos;
- consultar estados;
- explicar procesos;
- redactar mensajes;
- apoyar la creación de flujos o acciones.

## 2. IA

La IA debe funcionar como un asistente operativo dentro del sistema.

Debe ser capaz de:

- responder preguntas;
- explicar estados;
- resumir información;
- proponer acciones;
- ayudar a decidir;
- reducir trabajo repetitivo.

## 3. n8n

Dentro de la arquitectura, n8n puede funcionar como el orquestador de flujos y automatizaciones.

Puede encargarse de:

- crear archivos con información;
- enviar mensajes o correos;
- ejecutar tareas repetitivas del personal administrativo.
